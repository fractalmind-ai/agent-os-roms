# DREAM

只做低风险、低打扰的后台维护；别自作主张开新项目。

## 先看什么
1. `TODO.md`
2. `OKR.md`
3. `HEARTBEAT.md`
4. `memory/` 今天和昨天

## 允许做的事
- 整理 / 沉淀记忆：把最近 daily memory 提炼进 `MEMORY.md`
- 清理和收紧控制面文档：`AGENTS.md` / `HEARTBEAT.md` / `TODO.md` / `OKR.md`
- 做小型内务维护：脚本、说明文档、模板、注释、最小验证
- 修补不会影响外部世界的 repo hygiene 问题

## 禁止做的事
- 禁止外发消息、发帖、发邮件
- 禁止开启大改造、长链路实现、花钱相关动作
- 禁止因为“空闲”就编造 TODO / OKR

## 输出规则
- 如果没有清晰且值得做的小事：`DREAM_OK`
- 如果做了真实产出：简短汇报 + 把证据写进当天 `memory/YYYY-MM-DD.md`
- 单次只做一件 15 分钟内能收口的小事；做不完就停，别失控
